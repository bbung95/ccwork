# 타이포그래피

텍스트 크기·굵기·행간·자간 스케일 정의.
폰트 관련 스타일 작업 시 이 파일을 참조한다.

---

## 기본 설정

- **서체**: `Pretendard Variable` (fallback: `system-ui, sans-serif`)
- **기본 자간**: `-0.01em` (제목은 `-0.02em ~ -0.03em`으로 더 조인다)
- **단위**: `px`

---

## 스케일

| 역할             | 크기      | 굵기    | 행간 | 자간    |
| ---------------- | --------- | ------- | ---- | ------- |
| 앱 타이틀        | 21px      | 700     | —    | -0.02em |
| 에디터 제목 입력 | 23px      | 700     | —    | -0.03em |
| 카드 제목        | 17px      | 700     | 1.4  | -0.02em |
| 본문 (카드)      | 14.5px    | 400     | 1.62 | -0.01em |
| 본문 (에디터)    | 15.5px    | 400     | 1.7  | -0.01em |
| 버튼/인풋        | 15px      | 600     | —    | -0.01em |
| 메타·라벨        | 12.5–13px | 500–700 | —    | -0.01em |

---

## Do / Don't

**Do**

- 역할에 정의된 크기·굵기·행간 스케일을 그대로 사용한다.
- 기본 자간 `-0.01em`은 모든 텍스트에 적용하고, 제목만 더 좁힌다.
- 굵기는 400 / 500 / 600 / 700 네 단계만 사용한다.

**Don't**

- 테이블에 없는 임의 크기(예: 16px, 18px)를 새로 도입하지 않는다.
- 본문에 굵기 700을 사용하지 않는다 (제목 전용).
- `letter-spacing: 0` 또는 양수 자간을 사용하지 않는다.
