# 레이아웃

반응형 메이슨리 그리드, 상단바, 섹션 순서 규칙.
그리드·반응형·레이아웃 구현 시 이 파일을 참조한다.

---

## 반응형 컬럼

| 뷰포트       | 컬럼 수 |
| ------------ | ------- |
| `< 600px`    | 1       |
| `600–919px`  | 2       |
| `920–1119px` | 3       |
| `≥ 1120px`   | 4       |

---

## 규칙

- 콘텐츠 최대 폭 `1120px`, 좌우 패딩 `24px` (모바일 `16px`).
- 카드는 예상 높이 기준 **가장 짧은 컬럼**에 배치해 균형을 맞춘다 (메이슨리).
- 카드 간격(gap): `18px`.
- 상단바는 `sticky` + `backdrop-filter: blur`. 스크롤 시 하단 보더(`--line`)가 나타난다.
- **고정(핀) 메모**는 "고정됨" 섹션에 먼저 모이고, 나머지는 "메모" 섹션에 최신순 정렬.

---

## Do / Don't

**Do**

- 컬럼 분기는 위 뷰포트 기준점을 그대로 사용한다.
- 카드 배치 로직은 메이슨리(최단 컬럼 우선)로 구현한다.
- 핀 메모는 항상 일반 메모보다 앞 섹션에 표시한다.
- 상단바에 `position: sticky`를 사용한다.

**Don't**

- 컬럼에 고정 px 너비를 지정하지 않는다 — 컨테이너 폭에서 gap을 뺀 값으로 계산한다.
- 카드 높이를 수동으로 고정(`height: 200px`)하지 않는다.
- 상단바를 `position: fixed`로 구현하지 않는다 (sticky 사용).
