# 모션 · 아이콘 · 인터랙션

트랜지션·애니메이션·아이콘 규칙과 키보드 인터랙션 정의.
애니메이션 구현, 아이콘 추가, 인터랙션 설계 시 이 파일을 참조한다.

---

## 모션 기준표

| 항목                     | 지속  | 이징                            |
| ------------------------ | ----- | ------------------------------- |
| 카드 등장 (`card-in`)    | 0.32s | `cubic-bezier(0.16, 1, 0.3, 1)` |
| 카드 hover               | 0.2s  | `ease`                          |
| 모달 등장 (`modal-in`)   | 0.28s | `cubic-bezier(0.16, 1, 0.3, 1)` |
| 시트 등장 (`sheet-in`)   | 0.32s | `cubic-bezier(0.16, 1, 0.3, 1)` |
| 토스트 등장 (`toast-in`) | 0.3s  | `cubic-bezier(0.16, 1, 0.3, 1)` |
| 버튼 active (`scale`)    | 0.12s | `ease`                          |

### 원칙

- **표면 등장** — `translateY` + `scale` + `opacity` 세 가지를 함께 사용한다.
- **상호작용(hover/active)** — 빠르고 짧게 (0.2s 이하).
- 모든 주요 애니메이션의 기본 이징은 `cubic-bezier(0.16, 1, 0.3, 1)` (spring-like).

### Do / Don't

**Do**

- 카드·모달·토스트가 등장할 때 `translateY + scale + opacity`를 조합한다.
- hover/active 트랜지션은 0.2s 이하로 빠르게 처리한다.
- spring 이징(`cubic-bezier(0.16, 1, 0.3, 1)`)을 모든 주요 등장 애니메이션에 적용한다.

**Don't**

- 트랜지션을 1초 이상으로 설정하지 않는다.
- `linear` 이징을 사용하지 않는다 (기계적인 느낌).
- `opacity`만 단독으로 fade-in/out하지 않는다 (위치 이동 없이 나타나는 건 어색함).

## 아이콘

- **크기**: 24×24 viewBox
- **선 굵기**: `stroke-width 1.8–2.4`
- **스타일**: `stroke-linecap: round`, `stroke-linejoin: round` 라인 아이콘
- **fill 사용**: 핀 활성 상태, 노트 로고 등 채움이 의미를 전달할 때만

사용 세트: `note` · `search` · `plus` · `close` · `x` · `edit` · `trash` · `pin` · `pinLine` · `check`

### Do / Don't

**Do**

- 모든 아이콘은 24×24 viewBox와 라인 스타일을 기본으로 사용한다.
- `stroke-width`는 컨텍스트에 따라 1.8–2.4 범위 내에서 조정한다.
- 핀 활성·로고처럼 채움이 상태를 명확히 전달할 때만 `fill`을 사용한다.

**Don't**

- 위 세트 외 아이콘 라이브러리(Font Awesome, Material Icons 등)를 혼용하지 않는다.
- 아이콘 크기를 16px 이하로 축소하지 않는다.
- 기본 상태(비활성)에서 채움 아이콘을 사용하지 않는다.

## 키보드 & 인터랙션

| 동작                    | 결과                                               |
| ----------------------- | -------------------------------------------------- |
| 카드 클릭 / `Enter`     | 메모 열기 (수정 모달)                              |
| `Esc`                   | 모달 닫기                                          |
| `⌘ / Ctrl + Enter`      | 에디터에서 저장                                    |
| 에디터 제목에서 `Enter` | 본문으로 포커스 이동                               |
| 검색어 입력             | 실시간 필터 + 매칭 하이라이트 (`--blue-soft` 배경) |
